wikiopencite::citescoop::Parser::Parser()

Construct a parser that has no filter and no options.

Parsers with no filter will return all citations no matter their type.

wikiopencite::citescoop::Parser::Parser(std::function<bool(const std::string&)> filter, ParserOptions options)

Construct a new parser with a filter and parser options.

Parameters
filter Filter function to apply to citation types. The function is passed a normalized string (lower case, additional whitespace removed) of the first argument in a WikiText citation (see Wikipedia - Citation Templates for more details).
options Parser options to configure parser with.

Parsers may optionally filter based upon the citation type they encounter. This may be useful if you wish to exclude all citation templates that aren't "citation book" for example.

wikiopencite::proto::RevisionCitations wikiopencite::citescoop::Parser::Parse(const std::string& text)

Parse a given input string to extract citations.

Parameters
text WikiText to extract citations from.
Returns Citation protobuf representations.

If a filter has been set for this parser, the return values will only contain any citations that match the provided filter.
